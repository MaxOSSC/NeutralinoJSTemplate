# NeutralinoJSTemplate
NeutralinoJS项目模板
Neutralino.js v3.13.0
Neutralinojs v4.15.0
### 安装
```bash
npm install -g @neutralinojs/neu
```
### 初始化项目
```bash
neu create <project-name>
# 国内会失败
# 下载模板解压到项目目录里面
```
### 开启调试
```bash
cd myapp
neu run
```
### 项目打包
```bash
neu build --release
```
